INICIO

¡Te damos la bienvenida a MathLabInt!

¿Qué es MathLabInt?

MathLabInt (Laboratorio de Matemáticas Interactivo) es una plataforma educativa innovadora que transforma conceptos matemáticos abstractos en experiencias tangibles y visuales. Utilizando tecnologías como micro:bit y Arduino, creamos dispositivos que permiten "tocar" las matemáticas.

El Proyecto

Nuestro objetivo es construir dispositivos interactivos que visualizan funciones, geometría, estadística y álgebra en tiempo real. Cada módulo permite experimentar, formular hipótesis y validar resultados matemáticos de forma práctica, integrando programación, electrónica y matemáticas para desarrollar pensamiento lógico de manera innovadora.

Objetivos del Proyecto

🎯 Visualización Interactiva

Crear graficadores de funciones con matrices de LEDs que muestren comportamientos matemáticos en tiempo real.

📊 Simulación Práctica

Desarrollar simuladores de probabilidades y calculadoras estadísticas usando datos reales del entorno.

📐 Geometría Tangible

Implementar medidores de formas geométricas con sensores para comprender conceptos espaciales.

🤖 Integración con IA

Utilizar inteligencia artificial para generar ejercicios personalizados y adaptarlos a la realidad física.

TEÓRICO

Hardware

¿Qué es micro:bit?

El micro:bit es una pequeña computadora programable diseñada para hacer que aprender y enseñar sea fácil y divertido. Incluye sensores, pantalla LED, Bluetooth y conectores para componentes externos.

Arduino Uno

Plataforma de hardware libre basada en una sencilla placa con un microcontrolador. Ideal para proyectos interactivos que requieren múltiples sensores y actuadores.

Componentes del Proyecto

🔲 Matriz de LEDs 8x8

Para visualizar gráficos de funciones matemáticas

📡 Sensores

Ultrasonido, temperatura y luz para capturar datos del entorno

🎛️ Controles

Potenciómetros y botones para interacción del usuario

📺 Display LCD

Pantalla 16x2 para mostrar resultados y menús

Software

Utilizamos MakeCode para programar los micro:bit, Arduino IDE para Arduino, y herramientas de inteligencia artificial para generar contenido educativo personalizado. La programación se realizará principalmente en bloques visuales para facilitar la comprensión.

Disciplinas Integradas

Este proyecto combina matemáticas, programación, electrónica, diseño y pedagogía. Fomentamos el aprendizaje STEAM (Ciencia, Tecnología, Ingeniería, Artes y Matemáticas) de manera práctica e interdisciplinaria.

Materiales Utilizados

Microcontroladores:
2 micro:bit + 1 Arduino Uno
Visualización:
Matriz LEDs 8x8, Display LCD 16x2
Sensores:
Ultrasonido, temperatura, luz, acelerómetro
Actuadores:
Servo motores, buzzer, LEDs básicos
Componentes:
Resistencias, potenciómetros, botones
Conexiones:
Protoboards, cables de conexión

Reflexión Final

MathLabInt representa nuestra visión de hacer las matemáticas más accesibles y emocionantes. Creemos firmemente que cuando los estudiantes pueden "tocar" y experimentar con conceptos matemáticos, el aprendizaje se vuelve más significativo y duradero.

Este proyecto no es solo sobre tecnología, sino sobre transformar la manera en que entendemos y enseñamos matemáticas. Cada LED que se enciende, cada sensor que responde, cada dato que se visualiza, representa un puente entre lo abstracto y lo concreto.
